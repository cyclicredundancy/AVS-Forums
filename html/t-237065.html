glancing at code i see that youre coding inefficient.
keep in mind that a multiplication is much faster than a division so try to replace something like:
vara=varb/varc;
vard=vare/varc;
varf=varg/varc;

with:
invarc=1/varc;
vara=varb*invarc;
vard=vare*invarc;
varf=varg*invarc;
