This is excellent for a first pack. You have some sweet looking effects in there. You need to learn to get all the effects you want through the least things possible though. Like rotations. Instead of drawing all the scopes and using a DM to synchronise them. Use the same getosc functions and move them inside the scope. I can see you experimenting with simple DM's and scopes and this is good work. I really like the star one. Looks great. Overall i would give it a 3.3/5. Great noob pack although a little small, keep it up.
