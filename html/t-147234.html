use winamp 2.x for starters - it's less buggy and there's nothing wa3 does that 2.x doesn't afaik. also look for avs version 2.5.1...2.6.0 is, unfortunately, less user friendly and more bug ridden.

or...and i think this may be your problem...go to start menu -> run -> dxdiag, and check to make sure directdraw is enabled and running properly (under the display tab). i assume by static you don't mean actual random dots on the screen, but those random looking slanty lines that stay in the same place vertically and only take up a little bit of the screen - if that's the case, and the above didn't work, then you should reinstall directx....

hope that helps
