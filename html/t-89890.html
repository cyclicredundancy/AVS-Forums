To return to the topic:

The relationship between sin and cos is: sin(a)^2 + cos(a)^2 = 1
So if you have one coordinate of a circle and you want the other, you simply use:

b = sqrt(1-sqr(a));

(rearrange the formula above)

However if you want to generate a sphere, you're probably best off using spherical coordinates. Given 2 angles 'a' and 'b':


x=cos(a)*sin(b);
y=sin(a)*sin(b);
z=       cos(b);


First, 'a' defines a point on a circle in the x,y plane, then you rotate it onto the z-axis using the angle 'b'.

For a point with radius 'c', simply multiply each coordinate by 'c'.
