Err, you don't know about codable effect lists? Check em out, you can code effect lists now! It's way cool :D
You can just render the background for the first X frames and then store it to a buffer from which the dm will read it out. I'd say rendering it in a effect list is even faster than using a picture + you get all sort of neat things along with it, like randomness. A picture will always stay the same, but with a still background in AVS you can always render it differently with randomness or even code a few different modes for it manually :up:
