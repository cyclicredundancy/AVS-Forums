Solution
I had the same problem with Channelshift. Go to C://Program Files/Winamp/Plugins/avs and delete it. The duplicate is an update or something but the default APEs are somewhere else. I'm not sure if it works with Picture II or not as I only have one of them.

Channel Shift is part of the Winamp 5 vis_avs.dll, so deleting the ape does no harm. Picture is part of the core AVS file, but Picture II isn't. For anything you get doubles of, you should delete the ape.

using winamp 5 with avs 2.81 you can delete the following APEs, as they're integrated:

* ChannelShift
* Color Reduction
* Multiplier
* Texer

to get an up-to-date (and compatible) version of the Picture II APE, visit unconed's website
