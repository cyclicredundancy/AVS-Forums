First you'll need to get an gif animator if you want to create your own one. I propose you to download a recent version of Ulead Photoimpact (7 or 8) it comes with a powerfull gif animator.

Then you can work out pictures and blend them or do movements and stuff.

If you want to do a AVS Avatar, use a Video Capture Plugin. Try to save the stream as gif animation, if not possible you'll have to convert it later on. Then load it into a gif animator to do some final adjustments (size & framreate) to make the filesize small enough to fit wa.com's specifications.

This should get you a good start, just ask if you need detailed help :D

here are a few tips:
assuming you dont have any GIF animation software and dont want to pay for it, search Ulead GIF animator at google.com
read the help file so that you can get around and be familiar with the working of the software.
then make a cool animation, or make an avs preset, capture it to an .avi file and import it in Ulead GIF animator.
that's how i made my avatar :)

ulead is a 15 day trial program, so make sure you make it within that time. or download any ther free software. i suggested ulead because it is way cool ;)
