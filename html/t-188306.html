SyncBot 3.1
The third generation of SyncBot allows to backup all kinds of files to a FTP-Server or a local drive. The files are being compressed to save both, bandwidth and diskspace. SyncBot can be run once, on Windows start or in custom intervals.

Templates make it easy to backup a lot of Winamp related files, including AVS, MilkDrop and SPS presets, Skins or the entire Plugins directory.
