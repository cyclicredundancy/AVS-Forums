Marquee (Render Object idea)
Just a thought, but...

You know how Windows has a "Marquee" screensaver that scrolls text across the screen? Well, Render>Marquee would work kinda like that, only it would scroll the name of the song across the preset each time a new song in the Winamp playlist started.

Additionally, you could select various color/shape effects, such as 3D Text, bordered text, the ability to have the color of the text shift (either dynamically in response to the music - such as having it be an inverted color zone in the preset, or static, the way a Ring or Simple cycles through colors), have sections of the playlist listing come on at a time (say, Artist, then Title or vice-versa) and perhaps have the text pause on-screen for a while so that people could read it.

And naturally, ther would have to be an option for continuous scrolling of the information or one-time-only scrolling on track start.

The configuration options for the Marquee might be as follows:

Color: [color box here, disabled if Mode is Invert]

Mode:
( ) 3D
( ) Invert (this would be the effect that would cause the text to be an inverted color zone on the preset itself)
( ) Shadowed
( ) Outlined
( ) Plain

[ ] Pause for X milliseconds
( ) between sections
( ) when all information is onscreen

[ ] Separate Marquees for artist and title

( ) Scroll once at track start
( ) Scroll constantly
