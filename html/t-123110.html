Unless you're running on a 26 GHz processor, it's not powerful enough to run fullscreen at a decent framerate (unless of course you're running a really really simple preset). But if you're running it at some 320x240 and hitting "run on desktop" it's going to run at the same resolution as it's running in the AVS window. It's probably a problem with your Windows background settings, or the desktop color settings, or something. Make sure you don't have a background image, and that you have "Set desktop to color..." enabled.
