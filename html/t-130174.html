elvissuperspiroupgradebyshape - Nothing special. Try making something of your own from the ground up instead of just taking something already done and adding colors that don't follow any sort of pattern.
Wild Circle Havyx3 - Nothing special - at least throw a nice effect on it. All it is is some ssc's rotating on the surface of a sphere with some stars.
Magic of the geometry final - Very weirdly moving ssc, that would be alright if you made the colors a little nicer and added a nice effect to it.
Vertical Audio Scanner - Don't like. It's just a moving line with weird colors. Add effects to make it somewhat interesting.
Mind Out Of Limits - SSC isn't terrific and the effect isn't much. Apparently I don't have the font that you have. If you want others to see it the same as you, you are going to have to package the font with the presets and create an NSIS installer that will install them.
Time to make a life - Anyone here could tell you that I would absolutely abhor this preset. Moving Particles are very rarely used in a way that pleases me, and this is by far among the worst. The Dynamic shift is just the default one - nothing done to even make it better. Sorry, but I hate this preset.
Mind Out Of Limits 2 - Almost a decent preset. Try to use better colors, and add a more interesting effect - perhaps a DM.
Mind Pressure - Needs better colors and a better effect.
Feel the box - Synch the cubes and add a nice effect to it.
Overall: 1.75. You really need to choose better colors - a simple change from red to green to blue like that doesn't look that good and adds way too many colors with some of your ssc's. Also, learn how to code DM's - they will help you make much better effects. These presets aren't the normal n00b presets, but I wouldn't consider them to be better than some of the normal n00b presets. They are almost all based on the same SSC, and they aren't eyecapturing.

It's ok to revive threads if it's his thread and it's still on the subject. Which it is. DOn't critisise members unless you know what to critisise.

As for the preset. It's a little boring but i can see experimentation with scopes which can't be bad. Those scopes look weird and all the code doesn't seem to do much. It just seems to rotate a little circle of lines around 3d very slowly. Try changing x1, y1 and z1 to different things to get different results. eg for a sphere:

x1=sin((i*tpi*50))*(cos(i*pi+pi/2)*0.5);
y1=cos((i*tpi*50))*(cos(i*pi+pi/2)*0.5);
z1=cos(i*pi)*0.5;
(make sure you set the variables in the "init" box: "n=300; pi=acos(-1); tpi=pi*2;"

You've made (?) some 3d rotation and made crappy use out of it. Being able to code doesn't necessarily make good presets. You need to know how to make things look good.
