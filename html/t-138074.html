The answer is that AVS just is really slow! There is no way to magically boost it up. If you want fps, try some other plugin. But otherwise tone down the resolution.

I have a 1,73ghz amd athlon XP and i usually use 320x240 if i use fullscreen mode, 400x300 works fast enough sometimes, but nowadays avs presets are pretty complex to handle such "high" resolutions.

So make your window smaller and use smaller fullscreen resolutions. Always use 32bit color mode, because it works faster for avs.
