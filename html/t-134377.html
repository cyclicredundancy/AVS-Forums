Ok, sorry about the editing ;)

Some nice superscopes here, i think it's ok to use the El-Vis 3D engine, but many others on these forums don't think so...

Anyway, i think you are doing well with the coding part on the superscopes, but i think the some of the presets need trans effects as well...

Interpolation - Nice scope, and nice color coding. Tho i think it's a lot better with less fadeout and some other trans effects wouldn't hurt here either. Maybe a bit too minimalistic preset.

Bacround - Uhm... it seems you forgot to attach the picture in the zip file for this preset...

3DFireBall - This is definetly nice, but i think it would be better without the grow/shrink lines... Making the number of lines static would result in a better preset in my opinion. Also needs trans effects... And btw. using an effect list with Replace/Replace does nothing, it only slows down the preset. Don't stack up the effects inside an effect list if you're not going for an effect by it.

3dAbstract - Nice wavy shapes, but the same things bother this one as the 3DFireBall one. Including the unnecessary use of the effect list & line thingy.



Good to see some new avsers around here, and some good stuff too :up:
