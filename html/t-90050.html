Order of presets in a pack/presentation
I know that there are playlisters for professional presentation of presets during raves, etc. but it would be really nice providing a simple possibility for an avs-artist to order the presets in a pack or presentation using the u and y keys.

Idea:

A new misc module that could be added to the flowline of a preset specifying the next (u) and previous (y) preset combined with a reserved preset name like index.avs, startup.avs, etc. which is always jumped on first if navigating to an directory with u/y-key and which is always listed first in the presets listing of an directory.

If a specified preset or index.avs is not found the u/y key is switching to the next/previous preset as in the past.

If you make sure the PiMP or NSIS script includes each preset in the correct order, then they will extract in that order too. I've been using this since Whacko AVS pack :).
To save having to copy each filename into the script, you can number your presets and use wildcards ("01*.avs", "02*.avs", ...).
