From the current AVS wishlist
AVS Language Extensions

XOR, bit-negation and bit-shifting operators (^, ~, >>, << )

Add built-in constants like pi, 2pi, pi/2, e, ln(2), log2(e), ... into the language.

Document all functions (missing functions docs: atan2 and exp). Exp would probably not be used a lot, but atan2 is vital for conversion between cartesian and polar coordinates.

A full blown AVS language including flow control operations (repeat..until, while..do, if..then..else, case..of), commenting (/*..*/, //..CRLF), sub-routine definitions (function cube(x)=x*x*x), additional unary and binary operators (div, mod, shl, shr, inc, dec, etc.).

Combine all the event boxes (OnInit, OnFrame, etc) into one large syntax highlighting editor in which one would write each part as a subroutine. On top of this, try adding a feature so that AVS points out code errors instead of just choking.

A small list window showing all the variables used in a component's code (and their values).

Built-in support for imaginary numbers (you have to write your code to explicitly calculate with the real and imaginary parts of a complex number)


Usage and definition of global variables that can be accessed by all components of a preset.


A function that returns a real-time clock so that one can synchronize effects regardless of FPS or beat.
