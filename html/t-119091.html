NoBaddz (tm) (c) (r) (q) NSIS2 Installer script
This is a new installer script for NSIS2 that controls (extremely well) how the user can and cannot install an AVS pack, plus fonts and APEs. Instructions are contained within the .NSI file for how to edit the script to get rid of the code for APE and/or font installation.
* NOTE: Those instructions haven't been tested yet; I'm not totally sure they'll work. If there's bugs, e-mail me about them.
Included in the ZIP is Fonts.dll, a file needed for installation of fonts. I couldn't attach the working version of the installer (too big); you'll have to go to http://atero.deviantart.com/ to get it.

You may use installers made with this script AS LONG AS you give me full credit for the installer. If I catch you using this script without any credits, suffice it to say there'll be hell to pay.
