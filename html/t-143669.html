You can also run multiple instances of Winamp2. Right click -> Options -> Preferences and then in the Preferences screen Options and then check "Allow multiple instances". But I don't think you could run then in sync, unless you find a way to start the same song in both winamps at the same time. And ofcourse, both presets would be twice as slow.
