Pardon, JaVS, mais j'ai oublié tout mon Français.

"The Dog Ate My Reports" does get a little bright at times; that's due to the way the color effect works, which makes it tricky to fix without adding a bunch of stuff. I've been experimenting with tweaks, though; so far the best one seems to be adding "red=red*red; green=green*green; blue=blue*blue;" to the Color Modifier near the end of the preset. The preset becomes much darker, but still has bold colors... and it also gets bright at times, but not as much. This may be all the improvement that's necessary.

As for the "graininess" in "Entranced In Neons", I think I have a pretty good idea what you mean by that. A lot of the blame probably goes to the empty Effect List I use to smooth the animation a bit so the colors don't occasionally "strobe". I had the input set to "50/50" and the output set to "Every other pixel". That created an interesting effect, which I liked at first, but now I don't care for it that much. It probably looks better with the output set to "Replace". I also tried adding a Blur (Medium) before and after the Dynamic Movement to create a subtle fading effect and make the background look more like a background. It changes the effect a little, but I'm not sure if it's an actual improvement or not.

Currently, I'm working on a more colorful variation of "Portal Patrol", but I'm having FPS issues. I'll keep experimenting for a while, but I might end up needing to post the unfinished version here for advice on how to speed it up.
