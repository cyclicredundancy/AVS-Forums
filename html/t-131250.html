You didnt describe your problem properly so its difficlt to point out the problem, but heres what you can do:
1) Check whether a preset is loaded, if no preset if loaded then you will not see anything. Right click on the AVS screen and select any preset from the list.
2) Are you using fullscreen? On my pc, when I use Pixel doubling and resolution something X something X 32, I dont see anything.
3) Is the song playing? some presets are so made that you cant see anything if the song isnt playing. e.g UnconeD's energy ball.
4)If the above solution doesnt work, then try uninstalling winamp, then delete the winamp directory ( make sure you have installers for all the components you have downloaded ) and then re-install winamp again.

These are some of the common solution I could come up with. If it is anything to do with hardware, then sorry I cant help you.

P.S :- Make sure monitor is turned on ;) :p
