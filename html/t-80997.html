Problem using avs

Also the 848x480 fullscreen mode displays diagonal lines flowing across the screen, nothing resembling an a preset. I know the avs is the problem because when I disable it, winamp works fine. Found the problem!
If anybody else has this problem, the problem is with the mp3pro plugin I installed. I don't know why this plugin was causing the music to constantly start and stop when the AVS was on. All you have to do is delete the in_mp3pro.dll file from your plugin directory. Everything should work after that at least for me anyways! :p
