4th February 2006 17:17 UTC

Need a powerful CPU

CPU is avs main source of power so to speak. Upgrading this will help. Multi-core won't help much, but speed at floating point math will.

Don't need much memory

Yeah... AVS overhead is tiny... not even worth caring about. 8MB of system ram should be plenty for even the biggest resolutions

Don't need the latest graphics card

Again... not important a 4MB card, if they still made them would be good enough... unless you wanted to use resolutions higher than 2048x1536
