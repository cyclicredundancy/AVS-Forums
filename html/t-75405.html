AVS is a great visualization plugin. There are tons of presets out for it. In order to have AVS you must download either the full or standard version of winamp. It doesn't come with the lite version of winamp.

To use avs:

Winamp Preferences (CTRL + P)
Under the left column choose "visualization"
select "Advanced Visualization Studio v2.5 [vis_avs.dll]"
hit the start button
