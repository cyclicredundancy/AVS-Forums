i don't know why i have to keep saying this, but...
NEVER USE STRAIGHT UP GETOSC FOR STATIC RANDOMS. Use something like: (getspec(0.24,0,0)*10000)%100/1000-0.05
Basically it needs to move a LOT more, and change colors...it's a really cool effect, but it's ruined by that not-rotating thing. If you really need to keep what's in front in front, then use depth coloring and maximum blend.
