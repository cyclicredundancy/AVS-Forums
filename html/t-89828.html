This forum is only about AVS, a specific plug-in. If you wish to capture AVS's output into AVI, you can use an 'APE' I wrote called AVSGrabber (get it from http://avs.acko.net/). You'll need to insert it into the preset, so you need to have a basic understanding of how AVS works.

For other plug-ins, you'll have to use 3rd party applications that allow you to capture the contents of your screen (or capture it via Video out). Do a search in these forums for more info about AVS.
