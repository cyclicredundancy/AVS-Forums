Better than most first preset attempts,

tornado racer runs a little fast and is sort of a wash of red. At least add a colormap and/or define the scopes a little better against the background.

Use the force is quite fun but a bit of an experiment-preset, it needs to have some propper music response built into it to make it a propper preset.

Some of the code is a bit dubious too:

enabled=if(above(gettime(-1.0),gettime(-2.0)/2+3),1,0);

enable an effect list only if half the song + 3 seconds has passed? er.... ok.

trans/rotoblitter & trans/movement->tunnelling are essentially the same thing, you dont need both

Try not to use too many of the built in effects since they cannot be frame rate limited... and frame rate limit your movements becuase otherwise they will run too fast etc.. etc... generic noob advice... check the AVS tutorials for more
