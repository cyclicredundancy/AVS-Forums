I need to visualize the relationship between a list of timestamps and some music. I was hoping I could use the APE interface to accomplish this. Is there any way to find out during the render function what the current song time is?

You don't even need a new APE:

>> Start the AVS editor and remove everyting, enable "Clear every frame"

>> I don't know if AVS comes with "Misc > Global vars" by default - if you don't have it, add a "Render > SuperScope" instead.
>> Clear out all information in it (if you chose the superscope), write (or paste) "reg00=gettime(-1)" (without quotes) in the Frame box.
>> Add a Render > Text, and enter "$(reg00)" in there (again, without quotes) to display the exact playtime for the current track.
