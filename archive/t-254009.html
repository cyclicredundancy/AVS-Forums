i'm using wine (1.0) to run winamp (5.541) with avs (2.81b/avsmod 0.5.3 (http://www.visbot.net/vis_avsmoddll/index.bot)). both the winamp installer and pimpbot installers work perfectly and most of avs runs very well.

to give an impression of performance, i used the intro from visbot #16 (http://www.visbot.net/cat/VC016.zip) in 340x239 pixels (default dock-in-editor size). on windows xp (sp3), the preset performs at about 16.5fps. on ubuntu linux (8.04) i get about 14.5fps. not bad!

you can use winamp and avs as you're used to from windows, for example drag and drop of presets works the same. actually the (gnome) gui responds much faster than the native windows gui (e.g. menu fades)!

however, there are the same limitations as running avs in foobar2000 (http://forums.winamp.com/showthread.php?postid=2465005#post2465005): texer images will not display!

-wine (http://www.winehq.org/) (also available for mac os x (http://darwine.sourceforge.net/))
