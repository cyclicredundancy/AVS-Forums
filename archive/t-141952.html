fsk: those are just transformations that involve rotations and/or skewing, where the angle or distance depends on the coordinates.

A bend is where you rotate the coordinates in a plane, and the rotation angle is dependant on a coordinate along a vector in that plane.
E.g. bend an object in the XY plane, where the rotation angle for point (x,y,z) equals the x coordinate.

A twist/torsion is where the rotation angle in a plane is dependant on a coordinate along a vector perpendicular/normal to that plane.


Check the attachment and note the only difference between the two is the plane in which rotation occurs.

Miketrevis:

In the Trans / Dynamic Movement you will find in the Frame section these code lines:
xr=xr+xrs ;
yr=yr+yrs ;
zr=zr+zrs ;

xr=0 ;
yr=pi*0.5 ;
zr=0 ;

This will stop the rotation completely, if you want to rotate the tunnel end so it will be horizontally wider just use:

xr=pi*0.5 ;
yr=pi*0.5 ;
zr=0 ;
