You're opening it from Windows. It loads then closes before you can read it. You need to go to the command promt. This should be somewhere in your start menu.

Run it from there. eg.
cd\pimpdir
makepimp.exe

You'll then be able to see what it says.
Then type 'makepimp filename.psc' to create the exe file.
