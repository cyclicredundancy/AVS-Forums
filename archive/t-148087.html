montana - moonman mp (get it now)
yes, here it is, my latest release
it contains 5 presets, 2 new versions of old presetd, 1 remix of tuggummi's play blocks and 2 new presets

moondust: this is a new version of event:horizon, from mono3(i think), i gave it a diffent flavour, some are gonna like this other won't

starglow: this is a new version of way to io ffor it's going down, i was never satisfied with the version i created, so i stripped it down, added some new dm/alpha tricks and now i like it, the preset is a bit dark, but i like it that way

rest (when you're dead): this is the remix of play blocks, you will recognize the original, but i added a bit more annd gave it a diffent touch

thrill (my flow): this is a "alpha tricks" preset, basicly it's all about the alpha.

cloudwalking: it's a yet another 3d-rotozoomer preset with a twist,
hope you like it.

one last thins, i made this on a fast computer and i got almost 32fps on all presets, so if any presets are slow on slower computers, i apoligize for that
