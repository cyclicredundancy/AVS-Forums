I know this is kind of late, but I have tried this and it works. Set your screensaver to blank, and password protect it. Go to the display options on avs editor. Enable overlay mode and change the overlay color to pure black (0,0,0). Run avs when needed and turn on your screensaver. Tada! I find this is a good Idea for anybody with a simular problem in the future. Oh and be sure to change the overlay color back to default when you're done, so black text and stuff doesn't show visualizations on it.
