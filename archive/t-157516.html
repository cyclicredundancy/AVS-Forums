Hey all, at Brennan's request I've made AVS support SMP. There is an option in the AVS Settings/Display section to enable it, and currently only the following effects are accelerated: Movement, Dynamic Movement, Blur, Colorfade, and Brightness.

I will be doing more effects, as well as more tweaking and optimizing, but currently a lot of presets are faster on dual proc boxes. Woot.

For those of you who have 2+ processor machines, let me know how this works. If you have a P4 with HT, you can try and expect some slight performance improvements. If you want to try it and you have a single processor without HT, you can still try it to test and make sure it works...

http://firehose.net/~deadbeef/smp_avs_4.zip

-Justin

P.S. It might be worth turning off "Wait for retrace" and turning rendering performance to the highest to accurately measure framerate..
