You could install winamp to your 'windows/system32' directory, run it for the first time and set AVS to overlay mode, then leave AVS running but minimize winamp then quit it from the right click context menu from the system tray icon. Then find winamp.exe in windows/system32 and rename it to winamp.scr and set it to your screensaver that way when your screensaver runs it will run winamp, but it will be minimized so it will just display the AVS overlayed onto your desktop.
