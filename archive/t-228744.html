try changing the reg20 value which can be found somewhere down the frame box of the first superscope [it's the variable with the bigger text field next to it].
changing it to 0.2 will make a bigger rotation difference between the 2 scopes. maybe it's more easy to see then...

if this won't help reduce the reg11 value to sth between 0.9 and [minimum]0.4 to achieve more perspective distortion in each scope.
