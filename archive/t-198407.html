The standard output directory is the "../winamp folder". But nevertheless, you have to select an output format first. Otherwise the grabber doesn't know 'how' to save your data. If possible select any uncompressed(AVI) format from the list (later you can still convert it to mpeg/divX/...).
On my machine winamp often crashed when i tried to rip into certain compressed formats.
