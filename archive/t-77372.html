The thing is quite simple. if you want to get a hotlist loaded, select it from the list and click 'load'. this will load the complete hotlist into avs and you'll be able to activate the pre's you loaded up by the hotkeys you assigned them to. it's important that you activate the avs window first (click it) because avs only accepts hotkeys when it's focused. if you're tired of set a, select set b, load it and click avs again.

If you're wondering why you've to click 'load' all time, let me explain: let's say you're a professional VJ and you have about 200 pre's you want to use. You wouldn't assign your pre's to the shift+... Hotkeys because that's anoying, so you've 10 hotlist each with 20 pre's and screenshots of them. as you've loaded them you may whish to preview some other hotlist to find another pre that suits well. as this is done by selecting you'll have to click load.

some new important improvements...

I worked the whole weekend to finish them, now it's done!

First, there's now a direct load option. It simply loads the selected hotlist, you won't have to click load, if you don't want to.

Then, i added a Hotkey passing-through feature. That simply means, if you're working in VJ-Mode, every hokey you press will be directly passed to AVS (if running).
It works with all hotkeys, even the next/prev track or return for toggling windowed/fullscreen mode.

I tought that it isn't very usefull if you're doing a live set and suddenly you're pressing return by mistake... That's why i added a safety mode, that disables all non preset loading Hotkeys.

Furthermore i added a feature that allows you to work completely without hotkeys. If you click an item in the detail view window, first the selcted Hotlist will be loaded (if it wasn't already) and then, the clicked preset wii be started.
Unfortunately, it doesnt work with the shift+... hotkeys. You could keep on using them the old fashioned way, but i would recommend you to simply not use them - create another hotlist instead.

Last but not least, i made Hotlist save your VJ-Mode settings. If you have been using older versions of hotlist already please open the hotlist.set file in you Hotlist directory and add the following code on top of the file:

Detail=off
Pictures=on
Size=small
DirectLoad=off
SafetyMode=on

If you don't, it could cause Hotlist to crash o you're hotlists to be lost, maybe both.
