If you want to use a device connected to the linein port do the following:

1.) click "Open URL" and write:

linein://

2.) go to winamp preferences (easiest: press ctrl + P), search for "input" and configure "Nullsoft CD/LineIn Plugin": make sure that "sampling input from soundcard" is checked.

For the microphone channel i don't know the correct opening command :(
