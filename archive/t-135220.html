Not any automatic solutions anyway... you can use the keys Y & U to switch between PREVIOUS & NEXT, this should make it easy to shuffle trough a pack :)
But you need to make sure that the Settings -> Presets/Hotkeys -> "Preset (sub) directory to use for random/cycle" is either the pack you want to shuffle or ALL. I recommend you use ALL, because then you can just load the first preset from any directory (pack) and then just cycle them trough with the keys.
