You're referring to the player description I suppose?
I think that was a general description of AVS - there are thousands of presets for it to download.

There is no specific preset that they're referring to.
To get a stroboscope effect in AVS, select and start AVS:
Preferences (CTRL+P) > Plugins\Visualization > select "Advanced Visualization Studio v2.82 [vis_avs.dll]"
and click [start]

Right-click the window that opened and select 'AVS Editor'
select 'main' and click [-]
the list is now empty.
Select main again
Disable 'Clear every frame'
click [+] and select Trans\Invert
here's your stroboscope

to get rid of those horizontal lines, select settings > Display and enable 'wait for retrace'.

go [here] to download more (better) presets.
