You can't run overlay mode from WS mode, sorry.

You can set AVS to run when a song starts though.
Go to Options > Preferences (CTRL+P) > Plug-ins and enable 'Auto execute visualization plug-in on playback'

You could minimize winamp though, Overlay mode should still work.
You can control winamp (even when minimized) using Nullsoft Tray control (http://www.winamp.com/plugins/details/184)
