Using Windowshade and AVS
Usually I use Winamp in windowshade mode with the Bento skin and everything is perfectly fine. Recently I started using the "Overlay Mode" feature of AVS and I enjoy it very much. The only issue I have with it is that when I switch the window back to the Media Library or put it in windowshade mode it stops the AVS plugin and gets rid of the overlay. Is there a way I can just run AVS as soon as winamp starts up and just leave it running regardless of the visualization window being open or if it's in windowshade mode?

Warrior of the Light

You can't run overlay mode from WS mode, sorry.

You can set AVS to run when a song starts though.
Go to Options > Preferences (CTRL+P) > Plug-ins and enable 'Auto execute visualization plug-in on playback'

You could minimize winamp though, Overlay mode should still work.
You can control winamp (even when minimized) using Nullsoft Tray control
