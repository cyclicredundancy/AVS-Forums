For all the kranked-up AVS preset makers, this should be cheering news :)
The visualisation is called ZGEviz, based on ZGameEditor. You can make and edit presets in ZGE and import them in ZGEviz. If you aren't a owner of FLstudio, you can buy it via this link:
http://affiliate.image-line.com/BCJBAFB474
Otherwise, you can freely work with zgameeditor itself, but half the pleasure will be missing :P
Also i am really curious, i'm no coder, but i'm pretty sure that some parts of AVS could be inserted in ZGE. :)

ZGE is able to import 3dsmax models, any 3d models can be converted to 3dsmax format, too with external converters. It is a developper environement, you all will love it :)

If you already own FL9.9 BETA, you'll get it along with the software :) Otherwise, when you buy FLstudio producer edition, you are allowed to participate to the beta testing :)

Enjoy :)
