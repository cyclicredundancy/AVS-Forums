Close Winamp

Find your winamp.ini file (for me it's in D:\program files\winamp)

Open it and search for the line [AVS]

You should see some lines like these (adjust to taste):
wx= [x position]
wy= [y position]
ww= [width]
wh= [height]

I can't see what the fullscreen toggle line is, but I'll assume it's one of the cfg_fs_ ones.

Once you've fixed up the file, save it and run winamp again.
