Electric Puddle - If you don't know what something does then don't use it. This is my philosopy anyways. Not bad of a preset, and you did use some of your own code in the movements. The DDM really ruins it IMO. If you need help with DM's and the like search the forums or try pming someone(I will help you start out if you need it).
Fire Flower - Thank god that the rotating stars are hidden under the movements and interference and stuff. I would reccommend making a SSC instead of using the built in [crappy] renders. The interference also makes this extremely repetitive.
Flash Canyon - The SSC is extremely chaotic. Try something that is less chaotic. Also, try using alpha blending to hide the stuff in the distance, and possibly help conceal the sharp edges(this is advanced, though, so you can put it off until you learn more about it). Also the static movement inside the effectlist just creates horizontal lines completely defeating the purpose of using any sort of DM. It also makes the preset look worse overall.
Mirror Room - I do not like moving particles or rotating stars. Use something else. Also, replace/replace effectlists don't do anything - just bring the stuff outside of the effectlist. Instead of using a static movement for the rotation, try using a DM that will cause it to continue to rotate, and change directions/speeds(if you need help, search the forums/ask someone).
Rainbow Sea - The solid colors above the "water" really detract from this preset, but the vertical and horizontal lines are blaringly obvious and really hurt the preset. They appear because the superscope is very concentrated along those lines. Try using a different superscope that moves around some and doesn't stick out as much.
Spinning Wheel - Don't use random SSC's - they can cause Winamp to crash(it crashed mine anyways). Also, don't multiply r times any fractions/decimals - that's what causes that horizontal break in the movement. Once again, I don't like mp's when used like this - they are very hard to use well. Also, DM's would help make this better.
Stuck in da Flash Tunnel - Don't like flashing presets. Don't like MP's. Don't use replace/replace effectlists. Don't use random DM's. Do learn some more on how to code DM's.
Color ripples - MP's again. This is just a bunch of built-ins thrown together. Try to come up with something that keeps this from being like half of the other presets out there.
Overall: 2.25. Definitely learn how to code a bit better. Try not to use a lot of built-ins, and try to only use what you understand. Do this and you will improve. Not bad for a first pack.
