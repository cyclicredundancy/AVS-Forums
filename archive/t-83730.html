UnConeD
5th April 2002, 16:39

If you use Line-in, you won't be able to use the "advanced" beat detection. It attemps to detect the beat of the song and once it gets a close enough approximation, it locks into that beat. When the next song is played, it resets again. However, when using line-in, AVS will think it's still one song playing, so it will never re-detect the beat. Standard isn't bad though.
