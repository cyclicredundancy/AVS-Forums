Please include the APE's when you post your presets.

Anyways, here is my review:
running - Extremely slow. Try to optimize your code for speed. Don't like the moving particles(MPs) or the text. Try to find a more original alternative.
shaftoflight - Very slow. Once again - OPTIMIZATION. Don't like the MPs. Slightly boring as well - not much happens.
something good i hope - Very very very repetitive. You have to make the SSC be more dynamic.
vine-like - Interesting effect at first, but then it gets old.
cell membrane - Try using something other than an MP. Do that and this could be a nice preset. Oh, and don't use a pink background(looks much better with the invert enabled). Also try to speed it up some.
dynamic - Weird title considering that this preset is not dynamic - it does the same thing over and over again. Try to make it dynamic.
Earth&Sky - This idea has been done many times before. They were all done much faster as well. You need to make this one unique and speed it up. This preset just showed me something cool - if you use the ampersand in the filename of the preset, then AVS will give the preset the next letter as the hotkey like with windows programming - that's very cool.
leafycells - Background is static - make it morph or something. Don't use MPs in this manner - it gets really annoying.
Matrix - Boring. Seen the effect many many times before. Try something a bit more original.
MatrixSimple - Same as above.
notsureyet - Very similar to other presets of yours. Change some stuff and make the presets more interesting - this one does the same basic thing over and over again.
nullsoft - A black blob moving around leaving orange where it went. Not too interesting. Similar to others in this "pack".
Working on it - Not too bad. Try to learn how to get the camera angle to change - that will help make it more interesting.
water rings - A bunch of built-ins. Try to make something that doesn't rely solely on others coding(the SSCs and MPs, etc). Try to make a SSC or DM of your own to give it originality.
work in progress - Verrrrrry slow. Optimization needed. Not too interesting either.
Overall:2.5. Try to make your stuff more original, and don't use MPs and stuff nearly as much - I've just about drowned in all of the presets using MPs that all look basically the same - Try to differentiate yourself from this crowd of people. Learning some [more] SSC and DM coding could help a lot.
