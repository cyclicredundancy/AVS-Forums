Alright, I changed some things around here. There's a few things that doesn't change at all, like the blue box, the two back squares, and the black vertical lines for the sliders. Since they don't change calculating them every frame is a waste.

So what I did is this:

made a new effect list at the very top and put the things that wouldn't change in it.
added a buffer save at the end of the effect list and a buffer replace right after it.
changed the input and output blending both to ignore. disabled the effect list and enabled the evaluation override, then put "enabled=1" in the init section (this just makes the effect list enabled for the first frame only).


Now, after the first frame it just has to read the buffer to get all that stuff, instead of calculating it again every frame.
