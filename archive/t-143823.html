well the dot grid is kind of grainy and kind of ugly...there's better methods of giving it color. if you really want a monochrome background then you should try a dynamic clear screen:
superscope (init: n=0; point: x=0;y=0;) <- color this
movement (d=0)
to add it as a background you should put the rest of it in an effect list and use it as a render:
effect list (ignore, buffer 1 blend (might need to be inverted))
.preset...
.buffer 1 save (replace)
and then you should put the invert at the end so it doesn't go crazy.

hope that helped...
