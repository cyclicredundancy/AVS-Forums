If the latter, then the only thing that I know of that can be done with AVS text is to check/uncheck one of these 2 options for text, in the AVS Editor (to open Editor, double-click anywhere within AVS window)...

1. AVS Editor > Display > Windowed Settings...
a) check/uncheck Suppress status text in windowed
b) check/uncheck Suppress title text in windowed

2. AVS Editor > Display > Fullscreen Settings...
a) check/uncheck Suppress status text in fullscreen
b) check/uncheck Suppress title text in fullscreen
