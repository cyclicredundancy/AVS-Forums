Here's a new preset I made which was made to go with the Winamp 5 skin. It looks nice if you've got Winamp 5, if you don't, it hopefully still looks ok. Texer II was used a lot in it, and making this preset showed me just how much can be done with the Texer II. This and other recent efforts to use that APE are only scratching the surface of what can be done creatively. Anyway, to see how you can use it to make dynamic animations, try enabling the first effect list in this preset. You also may want to slow the preset down to see the frames better.

Oh and you definitely need Texer II and an AVS pre version to view this preset correctly.

EDIT: Don'y worry about the number of images in the Zip, they are all very small, in fact I think all except one are under 1kb. They ARE necessary for the preset, though.

EDIT2: The optional animation thingy with the Texer II won't be in the final release of this preset if I use it in a pack, which is why it is so shitty.
