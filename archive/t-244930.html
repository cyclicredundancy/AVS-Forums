Err, you don't know about codable effect lists? Check em out, you can code effect lists now! It's way cool :D
You can just render the background for the first X frames and then store it to a buffer from which the dm will read it out. A picture will always stay the same, but with a still background in AVS you can always render it differently with randomness or even code a few different modes for it manually :up:
