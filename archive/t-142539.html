pimpbot can best be described as a frontend for nsis 2 (http://nsis.sourceforge.net). within a few mouse-clicks it compiles an installer, which is custom-made for avs.

features
* autodetection of winamp 2+3
* visual customisation
* support for most common ape's

todo
* font support
* ?

please note that you need nsis 2 (http://nsis.sourceforge.net) to run pimpbot.

download pimpbot (http://visbot.net/index.php?title=TOOLS)
bugs? suggestions? comments? visit the forum (http://forum.visbot.net/viewtopic.php?t=17)!

PimpBot v1.1.0 RC2 (http://nsis.whyeye.org/pimpbot_1-1-0rc2.zip)
* updated layout
* supports multiple scripts
* additional/updated APEs
* new default script

still couldn't implement font-support, since i'd require a plugin for nsis which can read the font-name from a font-file.

also i want to point out, why people can't simply pick their own APEs or update the exisiting ones. the reason lies in the security i want to provide with pimpbot. if it had the features mention above, one could install APEs containing malicious code. using pimpbot is so to speak a quality/security predicate.

report bugs in the visbot forum (https://forum.visbot.net/viewtopic.php?t=17).

the final version will be released, once yet unknown bugs were eliminated and after the release of nsis 2 beta 4.

the current version requires nsis 2 beta 3 (http://nsis.sourceforge.net/site/index.php), it does not yet work with the current cvs or upcoming versions.
