Kalleidoscope : Too fast-changing to "enjoy" - An on-beat capture might help, just apply some transformations to keep it interesting.

Bouncy : The blurs do nothing and the texture is too abrupt. The DM also moves a bit too quickly.

Supersonic : Great work, but I don't think any of the blurs were needed.

Virus : Yuck.

Cult : Quite nice, the beginning movement doesn't exactly do much.

Floor Tiles : It looks very nice, but can use a lot of optimization. Put the mirror outside the effect list but before the DM.

Nightmare : Simplistic but stylish, I like it.

Rain : Neutral about this one. Find another way than using 4 heavy blurs.

Going Down : A replace/replace list is the same as no list as all. Add a fast brite, and to me this is a much better version of Virus, since, it uses the same "rendering scheme" (The low grid-size chaos)

Overall : 3.5 Stars, polishing them would likely to result in 4 stars.
