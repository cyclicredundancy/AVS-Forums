Hmm?
AVS doesn't overlay any song info like that, although presets for it can can.
Are you sure you're not using Milkdrop rather than AVS?

To add the artist + song name to an AVS preset, so it's always show you simply need to:
- Add a render -> Text
- Drag it down to the bottom (if it's not already, so it's drawn last)
- In the text to display box put: $(title)
- Fiddle with other text options to taste

But that will only affect the current AVS preset.
