For this review, I'll only mention the presets that you should take a second look at, whether it's because it's bad or good...

Lava Lamp - Delete blur, turn on Bilinear Filtering on Movement (Some speed improvement). It's quite static, but it's quite a good job for a beginner anyways.

Pref : Put fadeout on top, set normal size to minimum for Particles (And set "Default Render Blend"), delete the ones which aren't RGB (And duplicate one of them, change it to white, put it on the bottom of particles, put it on white, put it with slightly less onBeat change)
Replace the movement with a rotoblitter (And edit it as you desire).

Interference - Yikes, too chaotic... I like it a little better when the d*256 is off the first movement... Now, if there can be a better color-job...

Technicolour - Reminds me a bit too much of Lava Lamp...

Colour Pool - Not many people liked these kinds of presets...

Colorogy - Looks like Technicolour without the colorfade... You're using the firewurx a bit too much, don't you think?

Clouds and Cloudsmk2 - Didn't see much difference.

Solar Eclipse - There's a thin line between tranquility and boredom... This one's in the "static" group of presets... Hardly any movement.

Degauss - Great!
