Sheryas:
The code isn't the best, but here's what the variable names mean

sides    = # sides
rad      = radius of shape

height   = extrusion amount
h        = how far from the base shape plane (height)
rot      = rotation around the circle the shape is based on
p        = point

aa,ab    = how much to increment rotation in xy and yz
speed    = scale factor of rotation speed
xy,yz    = rotation in xy,yz planes

rcsb     = radius change beginning speed value
rcb      = radius change beginning value
rcssv    = radius change speed slowdown value
rcs      = radiud change speed
rc       = radius change
rcc      = radius change change (change the difference between the
           radii)
