Nice for a first try.. Keep on trying out components and see what everything does. Also, try looking into coded components and try if you can see what everything means, (var is just an example, you'd better not use 'var' as a variable; 'b' is a preserved variable that contains 1 if a beat is detected and 0 if not.) as for today you cannot work around coded presets anymore.
Get a tutorial somewhere if you can.

Also, bear in mind that you want to keep your presets to run as fast as possible, so try keeping out unused stuff out of your presets as it only slows things down.

By the way, I don't know if you selected an .AVI-file, but you have to add that into you zipfile as it won't be included into your .AVS file. This goes for Texer-images too
