Variables :
X - x axis
Y - y axis
I - Use to draw a line
v - Kinda hard to describe for me... sorta value of music (Think Simple/Oscillioscope)
w - Width of AVS screen (in pixels)
H - Height of AVS screen (in pixels)

You might want to learn the other commands, too - All the commands (sin, cos, tan, etc) from Dynamic Movement can be used for SSC...
