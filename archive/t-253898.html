Rockets game
I have finished my work on Rockets game - Stage 2 project.
Here is the pre-release of it in two archives.
Total zip size:~130 kb
Zip 1: preset, required GVM model, some atworks
Zip 2: another required atwork
Includes Keyboard function based on Grandchild's Text SSC, Keyboard is also linked with and used in Assistance (hint) mode.

Any feedbacks on the project are highly apperciated.
Please report if the patch was broken, and also if any other inconveniences or mistakes in preset are present.

If mission is complete, the stone position isn't reset after end of game (pershaps this appeared after the last edits, I didn't experience this earlier)
Inaccurate code of health meter (sometimes showing health<0)
Mistakes and inaccuracies in armor counter part of shot texer-II's, which make flying direction of last shot change after it is left when armor=0)
