From the Wishlist...

Add "current" to the buffer-list for Buffer Blending an effect-list. This would be like placing a Buffer Save at the end of your effect list and using that as blend-buffer.

I would also like to see a Current X2, X4, etc for ELs and Buffer Saves. Currently I always have to create a new EL (input only; no output) at the end of the original EL with a multiplier and a buffer save in it. I use this combination all the time and would like to see it in a more streamlined form, if possible. Mostly I use it for simulating fog type effects on top of other things (like in my preset "Swimming Through A Wormhole" in Energy). It's an extremely useful combination that could be made faster and simpler.
