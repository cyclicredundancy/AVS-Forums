Have you tried closing and restarting Winamp?

No go? Maybe this...

1. While Winamp is not running load the Winamp.ini file (from the Winamp directory) in Notepad or another text editor. Go to the [AVS] section and edit the following entries respectivelycfg_docked=0
cfg_cfgwnd_open=1
cfg_cfgwnd_x=0
cfg_cfgwnd_y=02. Save Winamp.ini
3. Restart Winamp
4. start AVS
