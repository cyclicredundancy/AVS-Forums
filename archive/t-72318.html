Sorry if you don't get a good framerate Montana. :( I designed it on a Pentium 4 1.5gHz and I get about 22 fps fullscreen at 800x600x32. If you have a slower computer, you are just out of luck. Any screen resolution below 640x480x32 produces serious problems in fullscreen mode (namely the problems that you listed). My suggestion is to run it in a window and try and get the best setup possible. If you have WindowsXP remember the alpha blending trick.

You are right about the whole effect list thing though. I used to have it set up with maximum blend on the nested list and then a fadeout set 80% above it. That changed before I released it but I never fixed the effect list. Thanks for catching that. :)

As for the second idea about loosing the first effect list... buy a better computer and it will run just fine. :D If you have trouble with both effect lists running, just turn off the first.
