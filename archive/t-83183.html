Umm...Those effects are directly from Sonique:

Zoom: Zooms every frame in
Zoom Speed: The distance that each frame is zoomed (drawbar)
Reverse: Zooms out instead of in
Smoke: This is essentially the same as y=y*1.01
Effects level: I believe that this defines how slow each frame is faded (drawbar)
Animation smoothness: The amount that each variable changes in the animation between menus and in the visuals.
Animation speed: The number of frames between menus.

Although Sonique's visuals can be a shitload better than regular AVS presets, hose functions don't make them any better or worse. That and the zoom effects, smoke, and effects level can be soft-coded, animation speed is essentially already available, and animation smootheness can't feasibly be transferred to AVS - and if it were, it wouldn't really make sense, as to change the smootheness, you would have to calculate the frames in-between; why not just render them??
