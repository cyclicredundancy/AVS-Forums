Hello,

Here's an idea. We all know that some presets are fast and some are slow. Some folks want complexity over speed and some want the thing to run at 30fps at a non chunky resolution.

How about a AVS preset efficiency index?

For example, I could set my AVS config so that preset that require above N calcs per frame would not be loaded. This way I could install whatever preset packs I wanted and might not see some of them but wouldn't be exposed to crawling unpleasantness.

I like variety and speed when I'm watching AVS on the TV. I'm willing to give up some of the more fantastic visuals to create a beat responsive and non chunky visual.

Comments? Suggestions? Flaming bags of merde on my doorstep?
