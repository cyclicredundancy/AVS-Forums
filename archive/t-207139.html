Warrior of the Light
7th February 2005, 13:53

first open AVS + the editor with the preset you want to capture

1) best place to add the APE is at the bottom of the list I don't know for sure if that's needed, just do it to be certain. ([+]>misc>AVSgrabber 2.0 alpha, drag to bottom)
2)select the outputfile. click [browse...] to select a folder and name your file. .avi is added automatically after your filename
3)select the playback rate (framerate). best is to leave both checkboxes on unless you want to do something else with your file.
4)select output format. select uncommpressed video or a codec from the list by clicking [choose...].
5)the .dll file is only for capturing music as well. for a website, this can be quite annoying. I personally didn't install the out_grab, but if you want to you can select it in the winamp preferences>plugins>output.
6)if you're done, hit [Grab!] to start grabbing frames and hit the same button to stop. the counter displays the frames captured
