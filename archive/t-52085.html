How do you start an AVS presert?
. - .... .- -. ....
9th June 2001, 18:26

Assuming that you have one installed, go to preferences (CTRL-P) -> plugins -> Visualisation and select the vis plugin that you want from the list. Depending on the plugin, the configure button will allow you to make/change settings for the visualisation. Then hit the start button.

If it's an AVS preset (not just a vis plugin), then double-click the vis in the main window. If you haven't messed with any other vis plugins, it'll pop up the AVS. Right-click the screen of the AVS, then hover over the folder you downloaded. Select a preset from the list that pops up.

If you have done what ethan said, then do what he said again, selecting "Advanced Visualization Studio"
