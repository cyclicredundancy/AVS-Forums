an automaton is any image that continually (and with relative predictability) morphs given a certain set of rules. conway's game of life is the most famous instance of a 2-d universal automata, another type of automaton is the ant automaton. some avs setups can be considered automata, given the changes to a pixel depend on the pixels around it and not where it is on the screen.
