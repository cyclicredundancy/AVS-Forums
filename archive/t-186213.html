Hmm... actually a quick shuffle through revealed that only the random preset cycling is answered in the FAQ not the "AVS pops up on the start of every song". But then again it's not entirely a AVS problem, see it executes the visualization plugin you've chose to run at the start of the song, which currently seems to be in your case: AVS.

To change this go to -> Options -> Preferences -> Plug-ins and uncheck the box "Auto execute visualization plug-in on playback"

This should kill it :)

My answer would be:

Q: AVS opens/doesn't open every time I start a new song.

A: You can disable/enable this setting by going to the Options > Preferences (Ctrl+P) menu.
Select Plug-ins (not Visualization) and disable/enable the option "Auto execute visualization plug-in on playback". (first checkbox)
